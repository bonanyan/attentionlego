## 模块介绍

本模块为DMA，主要负责两件事情：

1. 与外部存储器交互，获取例如Q、K、V权重的数据。
2. 连接内部模块，进行数据的传递。

## 模块架构

主要由3个通道构成：

1. 外部存储器和QKV计算模块之间的通道。会将总线上串行读取的数据转化为并行数据送给CIM模块。
2. QKV计算模块与QK乘积模块之间的通道。将计算得出的Q或K送给计算模块。
3. QK乘积模块与Softmax模块之间的通道。将计算的Score送给Softmax模块。
